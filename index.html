Webpack 学习总结


说起现在比较流行的框架，一定会想到vue react angular的天下了，但是说起优秀的前端模块管理工具，你一定会想到webpack,可以说现在最流行的这三个框架已经和webpack紧密相连了

Webpack 是什么


webpack是近期最火的一款模块加载器兼打包工具，它可以把各种资源，比如js，样式（less/sass）,图片等作为模块进行打包



快速搭建一个Demo


首先你要安装node和npm这里我就不介绍了，当这前期工作准备好了之后，
建立一个文件夹，比如叫webpack吧，
1. 安装全局的webpack npm install -g webpack
2. 初始化: npm init 在这个过程中会让你填name version 等等，如果开始学习的话，可填可不填，生成package.json文件，如下

3. 下载webpack: npm install –save-dev webpcak
webpack -v 查看一下webpack的版本号
4. 建立基本的项目结构
在根目录建立两个文件夹src文件夹和dist文件夹
src文件夹：用来存方我们编写的javascript代码，可以理解为代码源
dist文件夹：用来存放浏览器读取的文件，即打包的文件

让我们来聊聊webpack的配置文件


以上所说，在实际的开发中并不使用，而是使用webpack的配置文件进行设置
webpack.config.js

entry：配置入口文件的地址，可以是单一入口，也可以是多入口。
output：配置出口文件的地址，在webpack2.X版本后，支持多出口配置。
module：配置模块，主要是解析CSS和图片转换压缩等功能。
plugins：配置插件，根据你的需要配置不同功能的插件。
devServer：配置开发服务功能，后期我们会详细讲解。
